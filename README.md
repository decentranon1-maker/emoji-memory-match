# Memory Match - Emoji Edition

A classic memory matching card game with a 4x4 grid featuring 8 pairs of adorable emojis.

## Framework

Vanilla JavaScript (Static HTML)

## Features

- 4x4 grid with 16 cards (8 emoji pairs)
- Smooth card flip animations
- Move counter
- Win detection with confetti celebration
- New Game / Play Again functionality
- Responsive design for mobile and desktop

## Preview

Open `index.html` in any modern web browser to play the game.

## Gameplay

1. Click any card to flip it and reveal the emoji
2. Click a second card to find its match
3. If the emojis match, cards stay face up with a green highlight
4. If they don't match, cards flip back after 1 second
5. Find all 8 pairs to win and trigger confetti!

## Theme

Clean Light theme with soft white background (#f5f5f5), neutral grey card backs, and subtle green highlighting for matched pairs. Smooth animations provide a polished, premium feel.
